- `design.spec.md`（建議）
- `design.spec.txt`

- **實驗用**：從 RTLLM 複製 `design_description.txt` → `design.spec.md` 或 `design.spec.txt`

Workflow B 精煉結果 → `workflow-b-pipeline/spec_refined.md`（勿改 `spec/` 原始檔）。
